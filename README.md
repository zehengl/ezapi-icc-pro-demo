# parks-irrigation

![coding_style](https://img.shields.io/badge/code%20style-black-000000.svg)

A study on parks irrigation data

## Environment

- Python 3.9

## Getting Started

    python -m venv .venv
    .venv\Scripts\activate
    pip install -r requirements-dev.txt
    streamlit run streamlit_app.py

## Docs

    mkdocs serve
